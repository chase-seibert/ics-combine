# Agent Instructions

## Project Structure

- `combine_ics.py` - main script for combining ICS calendars.
- `tests/` - test suite for calendar-combining behavior.
- `calendars.example.toml` - example calendar source configuration.
- `Makefile` - project commands, including `make test`.
- `dist/` - generated build or distribution artifacts, if present.
